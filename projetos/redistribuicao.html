Detalhes do Projeto: Sistema de Redistribuição de Negócios

Sobre o Projeto

O Sistema de Redistribuição de Escolas automatiza a redistribuição de negócios entre SDRs (Representantes de Desenvolvimento de Vendas) com base em regras e limites. Ele integra o HubSpot CRM, armazena dados no Google BigQuery, envia notificações automáticas via Google Chat e trata negociações excluídas.

Tecnologias Usadas

  • Python
  • Google Cloud BigQuery
  • HubSpot API
  • Google Cloud Functions

Principais Funcionalidades

  • Integração com o HubSpot CRM
  • Armazenamento de dados no BigQuery
  • Notificações automatizadas via Google Chat
  • Processamento de negociações e geração de relatórios
